# Aeternitas

Protótipo estático para uma experiência interativa. Abra `index.html` no navegador.

## Acesso de teste

- Usuário: `Reality`
- Senha: `Cognitive`

O acesso é implementado somente no navegador, adequado para um protótipo. Em uma publicação real, o login deve ser validado por um servidor e as credenciais nunca devem ficar no JavaScript.

## Segurança

O botão da página Insania não baixa, instala ou executa arquivos. O projeto não implementa alertas falsos, coleta de dados, bloqueio de tela ou qualquer comportamento invasivo.
